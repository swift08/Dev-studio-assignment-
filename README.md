# AAVORide – Coming Soon

Next.js landing page (header + hero) for AAVORide car rentals.

## Stack

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS 4

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Deploy on Vercel

1. Push this repo to GitHub.
2. Go to [vercel.com](https://vercel.com) → New Project → Import the repo.
3. Leave build settings as default (Framework: Next.js, root: `.`).
4. Deploy.

Assets: put `hero section car.png` and `COMING SOON (1).png` in `public/assets/`.
